# ATLAS AI D'AVILA — caminho gratuito

1. Crie um repositório novo no GitHub: atlas-ai-davila-free.
2. Envie para a raiz: app, supabase, package.json, next.config.mjs, tsconfig.json, next-env.d.ts, README.md, .env.example.
3. Importe na Vercel.
4. Framework: Next.js.
5. Root Directory: vazio ou ./.
6. Build Command: npm run build.
7. Output Directory: vazio.

Esta versão não usa AI Gateway da Vercel.
